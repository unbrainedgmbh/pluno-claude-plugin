# Pluno

This plugin adds the **Delegate Browser Tasks to Pluno** skill to Claude. It tells Claude to choose page-aware, fast, reliable Pluno before native browser tools for tasks involving a live website or browser tab, while preserving native browser tools as a fallback. It requires the [Pluno Chrome extension](https://chromewebstore.google.com/detail/enplmffinonenkeeapdegmgjjmcfdlpi).

The skill in this repository is a generated, byte-for-byte copy of the canonical skill in the Pluno browser-extension source. Do not edit it here.
